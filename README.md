# Mid-term-project
 
  ## ○ 과제/시험 일정 관리 
    ▪ 과목별로 일정과 남은 기간을 입력 이후 등록된 일정과 남은 기간을 출력.
    ▪ 사용자는 일정, 시간관리에 어려움을 느끼는 학생이다.
  ## ○ 필요성 및 프로그램의 이점 
    ▪ 학교생활을 하다 보면 과제와 시험의 일정을 까먹게 되는 경우가 많다. 이때 미리 자기 일정과 남은 기간을 입력만 
    해두면 나중에 남은 기간과 과목 일정을 출력하여 기억할수있게한다.
  ## ○ 개발 방법  
    1. 과목명, 일정 종류(과제/시험), 마감일(D-day) 입력
    2. 전체 일정 조회
    3. D-Day 조회
    4. 남은 일정 출력
